一、对象（万能的、通用的、可以表示任何事物的） 1.对象的概念（object的实例对象） 在js当中，可以说一切皆对象； 对象的概念：无序的名值对的集合（键值对的集合）； 1.1 变量：如果存储一个简单的数据（一个数字，一个字符串），直接var a = 10; 1.2 数组：如果存储一堆的数据，此时我们想到数组，数组就是专门用来存储多个数据用的 1.3 函数：如果我们想要执行一段代码，或者让这段代码有功能，此时我们需要函数 1.4 对象：如果我想描述一个复杂的事物，比如说一个人，一台电脑（需要用到多个属性或者方法才能描述清楚）； 2.对象的创建方法 a) 字面量创建 var obj = { name:'zly', //name，age 这些被称为属性 age:32, sing:function(){}; //有动作用函数表达 } b) 构造函数New Object var obj = new Object(); 再调用参数把属性值传进去 c) 工厂函数模式 function createObject(name,age){ var obj = new Object(); obj.name = name; obj.age = age; return obj; } 3.对象的操作及遍历（增删改查） var obj = {}; var a = 'gender'; 增加： obj.name = 'zs'; obj["age"] = 30; obj[a] = 'male'; var a变量 → obj['gender'] = 'male'; 删除： delete obj.name 修改： obj.name = 'lisi'; 有则更改无则增加 查： console.log(obj.name); =↓ var name = obj.name; console.log(name); 读、写：方式，.语法，[]语法 。注意点：.语法和[]语法，还有变量 遍历：for in循环进行遍历对象 for(var key in obj){ console.log(key,obj[key]); } Document
