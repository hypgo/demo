函数执行的过程： 从第一句开始执行，发现函数定义了，不执行，当往下走发现有函数调用，就往上找函数去执行，最后去查返回值。 四、作用域 1、作用域 域就是范围 概念：作用域说的是变量起作用的范围； 作用：变量在各自的作用域当中起作用； 2、△局部变量和全局变量（两者没有任何关系） 局部变量：在局部作用域当中的变量（函数当中定义的变量）..理解为函数只是整个js的一小部分，就是局部 全局变量：在全局作用域当中的变量（函数外部定义的变量） 全局变量在哪都能使用，而局部变量只能在自己局部作用域当中使用（在函数中用）； 3、作用域分类(全局，局部，块级)， if(){ var a = 10 //在功能块中var的变量 } 在es5当中没有块级作用域，只有全局和局部。es6中增加了块级作用域 全局作用域和局部作用域当中定义变量的坑： 3.1当我们定义变量的时候，都会加一个var，但是也有不加var的情况： 3.2如果是在全局：没有定义过的变量，不加var，那么必须给这个变量赋值，而且相当于给这个变量加var，是一个全局变量。 3.3如果是在局部：没有定义过的变量（没有参数，也没有加var），首先要看外部全局是否定义过这个变量，如果全局定义过，那么这个 变量就不是在初始化，而是在操作全局变量(修改全局变量的值)。如果外部没有定义过，相当于在全局用var定义了这个变量； （如果有形参，相当于在功能块里定义了变量） 4、案例: 面试题：从上到下，一行一行走，碰到函数，不执行，一旦调用，开始执行。 var num = 10; function fun() { var num = 20; fun2(); } function fun2() { console.log(num); //10 } fun(); 5、作用域链：说的是变量在查找的过程，变量在查找的时候，首先从自己的作用域当中去查找，如果没有， 往上一级作用域当中去查找一直查知道函数外部的全局作用域，只要找到，立即停止往上找，直接输出。 如果没有找到会一直网上查，如果最后查到函数外部的全局都没有，就会报错； --> 外部不可以找内部变量，而内部变量可以往外部找。 6、△△△程序执行过程当中内存是怎么表现的。 栈结构：先进后出（水杯） 堆结构：任意增删 Document
